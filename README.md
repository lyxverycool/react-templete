## 本地开发
1. npm install
2. npm run dev

## 打包
1. npm i
2. npm run build

## 打包分析
npm run build:analyzer

### 全局lint
npm run lint

### tip
vscode 安装eslint插件 开启 Format On Save 在setting.json 中配置
  "editor.codeActionsOnSave": {
      "source.fixAll.eslint": true
  }
